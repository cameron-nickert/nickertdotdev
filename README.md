# Personal Portfolio

A Next.js + React + TypeScript portfolio foundation with App Router, baseline design tokens, and placeholder content.

## Local setup

```bash
npm install
```

```bash
npm run dev
```

Open `http://localhost:3000`.

## Scripts

- `npm run dev` - Start the development server.
- `npm run build` - Build for production.
- `npm run start` - Run the production build locally.
- `npm run lint` - Run Next.js linting rules.

## Project structure

- `app/` - App Router routes, layouts, and styles.
- `components/` - Shared UI components.
- `content/` - Structured content placeholders.
- `lib/` - Utilities and data access.
- `public/assets/` - Images, PDFs, and static media.
